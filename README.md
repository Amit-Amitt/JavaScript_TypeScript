## 🛠️ How to Run

### ▶️ JavaScript Files

1. Open any `.html` file in your browser
2. Open Developer Console (Right Click → Inspect → Console)

---

### ▶️ TypeScript Files

1. Install TypeScript:

   ```bash
   npm install -g typescript
   ```
2. Compile `.ts` file:

   ```bash
   tsc filename.ts
   ```
3. Run the generated `.js` file in browser or Node.js
